        <ul class="section-instructions-list">
            
                <li>
                    <div class="instruction-step">
                        <div class="instruction-number-box">
                            <div class="instruction-number">1</div>
                        </div>
                        <div class="instruction-content-box">
                            <div class="instruction-list-title">Step 1</div>
                            <div class="instruction-list-item"><p>Build the hardware:</p><p>- Make/order the DSI Shield PCB.</p><p>- Solder it!</p><p>- Check the BOM, as not all components should be mounted.</p><p>- Make/order/design an adapter board for your screen.</p><p>- Connect the two boards together with the LCD. Power the system through USB.</p><p>- No smoke indicates probable success.</p></div>
                        </div>
                    </div>
                </li>
            
                <li>
                    <div class="instruction-step">
                        <div class="instruction-number-box">
                            <div class="instruction-number">2</div>
                        </div>
                        <div class="instruction-content-box">
                            <div class="instruction-list-title">Step 2</div>
                            <div class="instruction-list-item"><p>Clone the repository:</p><pre>git clone https://github.com/twlostow/dsi-shield</pre><p>Install an LM32 toolchain. You may find one on Lattice website. There's a pre-built version for Linux (IA32) available <a href="http://www.ohwr.org/attachments/download/1133/lm32.tar.xz" target="_blank">here.</a></p><p>Download and install Xilinx ISE 14.7 Webpack from Xilinx website.<br><a href="http://www.ohwr.org/attachments/download/1133/lm32.tar.xz" target="_blank"></a></p></div>
                        </div>
                    </div>
                </li>
            
                <li>
                    <div class="instruction-step">
                        <div class="instruction-number-box">
                            <div class="instruction-number">3</div>
                        </div>
                        <div class="instruction-content-box">
                            <div class="instruction-list-title">Step 3</div>
                            <div class="instruction-list-item"><p>Build the software:</p><p>- edit software/rev1/panels.h and set the panel type you wish to use.</p><pre>$ cd software</pre>
<pre>$ make</pre><p>This will produce the boot.ram file that the synthesizer will embed in the FPGA bitstream. The file contains both the panel initialization/HDMI handling code and the bootloader, so the LM32 application can be re-loaded at any time via USB UART.</p></div>
                        </div>
                    </div>
                </li>
            
                <li>
                    <div class="instruction-step">
                        <div class="instruction-number-box">
                            <div class="instruction-number">4</div>
                        </div>
                        <div class="instruction-content-box">
                            <div class="instruction-list-title">Step 4</div>
                            <div class="instruction-list-item"><p>Build the FPGA bitstream:</p><p>Open hdl/top/rev1_top.vhd and edit the PLL configuration genrics to match your LCD panel. See Table 1 above the VHDL entity declaration for the values.</p><pre>cd hdl/syn/rev1; ise rev1.xise</pre><p>Click <em>Generate Programming File.</em></p><p>After a while, you'll get the bitstream: <strong>rev1_top.bit</strong>.</p><p>Program the SPI flash with the bitstream through JTAG.</p></div>
                        </div>
                    </div>
                </li>
            
                <li>
                    <div class="instruction-step">
                        <div class="instruction-number-box">
                            <div class="instruction-number">5</div>
                        </div>
                        <div class="instruction-content-box">
                            <div class="instruction-list-title">Step 5</div>
                            <div class="instruction-list-item"><p>Power-cycle the board. A test screen with the panel info and "No HDMI signal" message should appear. If it doesn't... well, something's wrong. Otherwise, connect the HDMI input. If everything is OK, the PC should detect the display resolution and automatically configure it.</p></div>
                        </div>
                    </div>
                </li>
            
        </ul>
